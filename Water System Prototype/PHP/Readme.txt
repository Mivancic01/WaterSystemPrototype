This is the WebGL build of the game which uses a local php server.

The php script is used for loading save files.

Instructions:
1. Before starting the game open the terminal within the folder where tounity.php is located.
2. In the terminal use the command "php -S localhost:9000"
3. Start the game normaly

Optional:
The game can be started without a php server, but then it wont be able to lad any save files.
